
# PHI.JS
phi.js는 webgl2 기반 렌더링 엔진입니다. 한별중학교 dev팀을 위해 개발되었었으며 재학생이라면 누구든지 무료로 사용가능합니다. 재학생이 아닐경우 허가후 사용가능 합니다.

# 기본세팅
기본 사용법은 아래와 같습니다.

1. `@phi` 폴더를 디렉토리의 가장 상위에 배치합니다.
2. `.html`파일을 만든후 `<canvas>` 태그를 삽입합니다. 이때 `<canvas>` 는 아래와 같이 작성합니다.
    
    `<canvas id="사용할아이이디"></canvas>`  를 필수로 작성해야 사용가능 합니다.

3. 스크립트 파일을 만든후 `html`과  연결합니다. 이때 `type=module` 을 `script` 태그안에
    
    필수로 작성해야 합니다.



##### 이후 `@phi`안 지침서에 따라 코드를 작성해주세요.
